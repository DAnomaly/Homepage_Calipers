KeyScreening
- 카드회원가입 심사업무 자동화 솔루션

키스크리닝(KeyScreening)은 신용카드회사의 카드회원가입 심사업무를 효율적으로 수행하기 위해 이미지 시스템과 워크플로우 시스템을 기반으로 카드회원심사 업무자동화를 구현한 솔루션입니다.

카드회원가입 관련 문서의 접수, 분배, 보정, 심사 관리에 이르는 전과정을 자동화함으로써 빠른 업무처리 및 대 고객 서비스가 가능하도록 합니다.

카드회원신청서에 대한 보정작업을 수행하고, CTI 연동으로 고객과의 직접적인 대화를 통한 심사를 지원하며, 심사에 필요한 다양한 정보를 연계하여 심사업무 담당자가 정확하고 빠른 판단을 바탕으로 업무를 수행할 수 있도록 지원합니다.

주요 특징

1. 검증된 시스템
- 전세계 금융기관이 선택하는 EDMS 엔진을 기본으로 탑재한 검증된 시스템
2. 시스템 확장의 용이성
- 작업의 과부하, 네트워크의 과부하에 따른 분산업무 환경 및 Multi Server, Multi System 구현
- 전자문서관리, COLD, 이미징, 워크플로우 제품을 단일 또는 복수로 구축 가능하고 향후 확대 시
- 유연한 통합 가능
3. 다양한 형태의 문서 수용
- Scan된 이미지, 동영상, 사진, Word, Excel, 녹취 데이터까지 종류에 상관없이 저장 관리
4. 대용량 고속 처리
- 1일 백만건 이상의 Transaction을 처리 할 수 있는 유일의 전문 금융 솔루션
- 처리의 긴급성에 따른 우선 순위 처리 기능
5. 완벽한 분산성 및 무정지 시스템 관리
- 완벽한 분산 서버 지원은 물론 원격지 제어 및 서비스
- 시스템 가동 중 설치 및 백업으로 무정지 시스템 관리 가능
6. 완벽한 백업과 Recovery
- 온라인 무정지, 백업 광디스크 자동 복제 및 복구
7. 업무 변경에 따른 신속한 적용
- 업무 변경 시 Process Map 수정(Visual Map)에 의한 변경업무 적용
8. 자동심사 강화로 인한 효율적인 업무 진행
- 심사 종류를 전문가 및 일반 심사 등으로 분류하고 일반 심사 중 조건을 다양하게 입력하여 자동
심사 처리를 적극 활용함으로써 효율적인 업무를 할 수 있도록 함
9. 완벽한 보안문서관리
- 고객의 주요 데이터 보완을 위해 DRM 기능 적용

주요 기능

1. 입력기능
- 청약서/신청서 및 관련 서류 이미지 스캐닝
- OCR/OMR 인식 기능
- FORM Processing
- 이미지 인덱싱
- 데이터 압축, 즉시/배치/예약 전송
2. 보정기능
- 보정해야 할 건과 보류, 취소, 완료 건 조회
- 지점에서 보내 온 스캔이미지로 본사에서 보정 투입
3. 심사기능
- 상태별 또는 신청일자별로 WF 심사대상을 검색 조회
- 심사 대상인 고객의 정보 확인
- CTI 장비인 PDS와 연동으로 자동전화연결 기능
- 현재 진행되는 심사고객의 전체 심사건 확인
- 고객의 신청카드, 기업정보, 직종코드 확인 및 한도산출 기능
4. 통계 및 검색 기능
- 업무별, 유형별, 지점별, 처리 이력별, 호스트 거래 조회
- 다중 조건 별 검색
- 보정자, 심사자 평가 통계
- 모든 형태의 문서를 어떤 환경에서도 접근 가능
- 계층적 분류에 따른 폴더, 본문 내용, 제목 등 강력한 문서 검색 기능
5. 자료 분배 기능
- 결재(결재/보류/반려/전결): 지급 승인자 결정 및 승인
- 접수~승인에 이르는 이력 관리
- 자필 서명 비교
- 자동심사 및 전문가 심사 등 단계별 심사 기준 적용
- 지점별, 보종별 심사 배정
6. 서버 모니터링
- 에이전트 서비스 모니터링
- MRII 서비스 모니터링
7. DRM 기능
- 사용권한에 따른 문서 접근 제어
- 인쇄 출력 및 외부로의 정보 전달 차단
- 저장 / 새 이름으로 / HTML 저장 차단
- Copy & Paste에 대한 차단
- 사용권한에 따른 Screen Capture 제한
8. 관리자 기능
- 사용자 관리, 코드 관리, 심사 자료 관리, 심사 기준 관리
- 심사자 관리(심사자 변경 및 Queue 이동)

적용 범위

1. 업무 프로세스 변경 및 설계 시

회원입회 심사, 연체관리 심사, 대출 심사, 민원관리 심사

2. 각종 서류 및 첨부문서의 이미지화

입회신청서, 법적조치 서류, 대출 신청서
