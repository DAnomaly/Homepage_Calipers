MobileWay

모바일웨이(MobileWay)는 기존의 데이터를 무선 네트워크를 통해 다양한 모바일 디바이스를 이용하여 서비스를 제공받을 수 있도록 만든 무선 애플리케이션 서버입니다.

모바일 SI를 통해서 무선인터넷 시스템을 구축할 경우, 기존 회사 내의 각종 데이터를 통한 이동 사무실 환경을 구현하기 위해서는 각각의 무선 애플리케이션을 별도로 구축해야 하는 반면, 모바일웨이(MobileWay)는 GUI 환경을 통하여 하나의 서버로 구현할 수 있어 업무 생산성 및 효율성이 매우 뛰어납니다.

다양한 무선 애플리케이션을 하나의 시스템으로 지원합니다.

주요 특징

1. 전문가 수준의 프로그래밍을 손쉽게 Tool을 이용하여 제작
- GUI 화면을 이용한 마우스 클릭으로 모바일 프로그램 제작 가능
2. 유무선 통합 지원 시스템
- 한번의 프로그램 개발로 모든 모바일 기기와 유선기기를 지원함으로써 개발 생산성 향상
3. Legacy System과의 강력한 연동성
- Groupware, DB, ASP, JSP 등 다양한 Legacy System을 지원함으로써 어떠한 정보시스템과도 손쉽게 정보를 교환할 수 있음
4. 개방형 아키텍처
- XML과 XSL을 이용한 개방형 아케텍처로 구성되어 있으며, Java를 기반으로 개발되었으므로 어느 OS에서도 설치될 수 있음
5. 기능별 Component 구성
- 시스템의 개방성과 확장성을 용이하게 하기 위해 기능별 Component로 구성
