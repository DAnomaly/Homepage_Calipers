Triple Print PRO

트리플 프린터 프로(Triple Print PRO)는 인쇄를 지원하는 Windows 응용프로그램을 통해
이미지 서버에 등록해야 하는 전자파일(HWP, WORD, PPT, HTML, 각죵 Image)을
출력 및 SCAN없이 간편하고 편리한 방법으로 이미지서버에 저장할 수 있도록 구현한 솔루션 입니다.

여러 개의 데이터를 동시에 이미지 서버에 저장하고 해당 이미지를 단일 된 확장자로 보관할 수 있으며
이미지 생성과 서버 전송까지도 일괄적으로 처리하여 사용자의 효율적인 작업을 보장합니다.

Triple Print PRO 업무 흐름도

  1. 문서
  2. 가상 이미지 출력
  3. 저장
  4. 전송
  5. 데이터 저장
  6. 조회

주요특징

  1. 인쇄를 지원하는 Windows 응용프로그램에서 이미지 서버에 지정해야 할 전자파일(HWP, WORD, PPT, HTML, 각종 IMAGE파일)을 출력하지 않고 단일화된 TIFF 또는 PDF 파일로 변환
  2. 40/128비트 PDF 암호화 지원
  3. PDF 선형화(linearization) 지원
  4. 글꼴 내장, 해상도 및 이미지 압축 지원
  5. Windows 탐색기에서 가상프린터를 통하여 PDF, EPS, PostScript, TEXT, JPEG, TIFF, PNG, PCX, BMP, PDS and PCL-X로 변환

주요기능

  1. 인쇄를 지원하는 Windows 응용프로그램에서 이미지 서버에 저장해야 할 전자파일을 출력없이 단일화된 이미지 파일로 저장 및 이미지 서버에 전송
  2. 전자파일 PDF, EPS, PostScript, TEXT, JPEG, TIFF, PNG, PCX, BMP, PDS and PCL-X 등 Windows 응용프로그램에서 인쇄가 지원하는 모든 확장자 지원
  3. 여러 문서를 단일화된 PDF 또는 TIFF파일로 결합
  4. 40/128 비트 PDF 암호화 지원
  5. 해상도 및 이미지 압축 지원, 내장 글꼴을 통한 동일한 이미지 지원
