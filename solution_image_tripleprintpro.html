Triple Print PRO

 트리플 프린터 프로(Triple Print PRO)는 인쇄를 지원하는 Windows 응용프로그램을 통해 이미지 서버에 등록해야 하는 전자파일(HWP, WORD, PPT, HTML, 각종 Image)을 출력 및 SCAN없이 간편하고 편리한 방법으로 이미지서버에 저장할 수 있도록 구현한 솔루션입니다.

 여러 개의 데이터를 동시에 이미지 서버에 저장하고 해당 이미지를 단일 된 확장자로 보관할 수 있으며 이미지 생성과 서버 전송까지도 일괄적으로 처리하여 사용자의 효율적인 작업을 보장합니다.

주요 특징

1. 인쇄를 지원하는 Windows 응용프로그램에서 이미지 서버에 저장해야 할 전자파일(HWP, WORD, PPT, HTML, 각종 IMAGE파일)을 출력하지 않고 단일화된 TIFF 또는 PDF파일로 변환
2. 40/128비트 PDF 암호화 지원
3. PDF 선형화(linearization) 지원
4. 글꼴 내장, 해상도 및 이미지 압축 지원
5. Windows 탐색기에서 가상프린터를 통하여 PDF, EPS, PostScript, TEXT, JPEG, TIFF, PNG, PCX, BMP, PSD and PCL-X로 변환
6. 여러 문서를 단일 PDF 또는 TIFF파일로 결합/매칭

주요 기능

1. 인쇄를 지원하는 Windows 응용프로그램에서 이미지 서버에 저장해야 할 전자파일을 출력없이 단일화된 이미지 파일로 저장 및 이미지 서버에 전송
2. 전자파일 PDF, EPS, PostScript, TEXT, JPEG, TIFF, PNG, PCX, BMP, PSD and PCL-X 등 Windows 응용프로그램에서 인쇄가 지원하는 모든 확장자 지원
3. 여러 문서를 단일화된 PDF 또는 TIFF파일로 결합/매칭
4. 40/128비트 PDF 암호화 지원
5. PDF 선형화(linearization) 지원
6. 해상도 및 이미지 압축 지원, 내장 글꼴을 통한 동일한 이미지 지원

적용 범위

1. Paperless office 구현 시
- 심사, 결제 등 다수의 종이서류 발생 시
- HWP, WORD, PPT등으로 이루어진 서류를 등록하기
2. Internet 자료 첨부하여 통합 관리 시
- Internet을 이용한 조회 결과 첨부하기
- 이메일 또는 메신저를 통해 받은 서류를 등록하기

○○보험
 보안을 위해 인터넷 망과 업무망을 따로 쓰는 환경에서 보다 효율적으로 이미지전송 프로그램을 사용해야 함으로 TripleAims의 Triple Print Pro를 이용하여 보다 쉽게 이미지 등록이 가능하도록 구현하였습니다.
  1. 인터넷 망 또는 업무망에서 등록이 필요한 서류를 조회한다.
  2. 서류 등록을 위해 가상프린터로 인쇄 명령을 내린다.
  3. 종이문서를 인쇄하지 않고 자동으로 이미지를 생성한 뒤 이미지관리 프로그램으로 전송합니다.
