KeyDocs

키닥스(KeyDocs)는 사내외 디지털 정보의 체계적 관리와 활용을 위한 차세대 전자문서관리(Electronic Document Management) 시스템입니다.

키닥스는 업무 단위별 문서 분류 체계를 적용함으로써, 사용자의 정보 접근에 용이성을 제고하고, 고객의 업무 효율성과 업무의 질 향상을 도모하였습니다.

또한, 컨텐츠 관리(Contents Management) 기술을 활용하여, 문서관리에 저장된 컨텐츠를 바로 웹으로 올릴 수 있도록 함으로써, End-User와 직접 Communication 할 수 있도록 한 성공적 지식경영을 위한 핵심 도구입니다.

선진 기업이 선택한 FileNET IDM Engine을 기반으로 하여 저장된 데이터 저장에 대한 안정적인 보호와 완벽한 보안을 자랑하고, 시스템 확장성과 유연성을 가지고 있어, 다양한 시스템의 변경 요구에 대한 빠른 수용이 가능합니다.

귀사의 탄탄한 지식경영 인프라 - 키닥스 (KeyDocs)가 함께 합니다.

주요 특징

1. 업무단위별 문서 분류체계

2. 강력한 시스템 통합 검색

3. 뛰어난 확장성(Scalability)
- 단순 작업그룹 차원에서 전사적인 분산 환경까지 동일 소프트웨어와 구조로 지원합니다.

4. Performance(고성능)
- 중요 시스템 자원을 효과적으로 사용하기 위해서 최적화된 알고리즘을 채택하였습니다.

5. Security(보안성)
- 문서관리시스템에 접근하기 위한 기본적인 사용자 인증은 물론 모든 문서 및 시스템 정보 접근을 위한 접근 권한 제어를 서버에서 수행함으로써 보다 안전한 시스템 접근 통제가 가능합니다.

6. 각종 통계정보의 강화
- 문서 종류별, 폴더별 등록현황 조회 및 평가현황 등 다양한 통계 정보를 제공합니다.

7. 컨텐츠 관리(Contents Management)를 통한 고객접점 기능 강화

8. 안정성 높은 FileNET IDM Engine 기반

주요 기능

1. 문서관리 기능
- 문서를 수정하기 위한 체크아웃 기능
- 문서의 체크아웃을 취소 기능
- 체크아웃된 지시를 수정하는 체크인 기능
- 지정된 문서의 보안 설정 기능
- 현재 문서가 소속된 폴더를 보고 수정 할 수 있는 기능
- 문서의 등록/삭제 기능
- 문서에 대한 버전 정보 기능
- 복합 문서의 첨부 기능
- 문서에 대한 인덱스 설정 기능
- 설정된 인덱스를 해제 하는 기능
2. 각종 관리 기능
- 삭제 문서 관리 기능
- 사용자의 체크아웃 한 리스트 검색의 체크아웃 목록 기능
- 현재 설정된 폴더 현황을 보여주는 폴더관리 기능
- 폴더 등록, 변경, 삭제, 복사, 이동 기능
- 보안에 대한 권한 설정 기능
3. 다양한 검색 기능
- 문서의 속성으로 검색하는 기본 검색 기능
- 문서 속성을 통해 검색하는 상세 검색 기능
- 등록 문서 내용을 검색하는 내용 검색 기능
- 등록된 폴더 검색 기능
4. 문서 등록 / 삭제 및 색인 설정/해제 기능
- 문서의 종류, 폴더선택 등 기능을 추가하는 문서 등록 기능
- 시스템에 등록된 문서를 삭제 하는 기능
- 내용 검색을 위한 인덱스를 설정 및 해제 기능
5. 컨텐츠 관리 기능
- Template: 웹 사이트에 적용할 홈페이지 포멧
- Converter: 기업 내외에서 발생된 문서를 전달 받은 파일을 웹으로 게시하기 위해 전환하는 변환 기능
- Workflow: 문서에 대한 결재 모듈을 통해 웹으로 게시한다.
- Scheduler/Publishing: 변환 문서를 스케쥴러와 퍼블리셔를 통해 웹 게시
- Multi-Channel Service: 유무선 장비들과의 연동을 통한 컨텐츠 서비스
