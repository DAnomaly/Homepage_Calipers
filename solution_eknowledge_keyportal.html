KeyPortal

키포탈(KeyPortal)은 EIP와 KMS의 기능을 통합한 기업지식포탈(Enterprise Knowledge Portal) 솔루션입니다.

사용자가 개인의 취향과 업무 필요에 따라 개인 포탈 화면을 자유롭게 구성하여(Personalization), 기업 내의 수많은 지적 자산을 체계적으로 관리, 전달, 활용할 수 있도록 한 성공 지식경영을 위한 확실한 Tool입니다.

각종 디지털 정보에 대한 지식평가와 마일리지 시스템이 통합 구현되므로, 사장되기 쉬운 기업 내의 노하우가 체계적으로 관리, 공유되는 지식경영 솔루션입니다.

주요 특징

- 키포탈(KeyPortal)은 애플리케이션 통합과 컨텐츠관리, 메타데이타 관리에 뛰어나므로 전사적 통합 솔루션 구축이 용이합니다.

1. 맞춤 정보 제공을 통한 가치화 된 정보 재창출
- 기업의 대부분을 차지하고 있는 비정형 정보의 통합을 통한 지식 LEVEL의 향상
- 지식정보의 완벽한 통합을 통한 효과적인 정보 이용 및 공유 가능
- 통일성과 일관성을 극대화 한 맞춤정보 제공(Personalization)
2. 타 시스템과의 원활한 연동을 통한 협업 체제 구현
- ERP,CRM,MIS등 기존의 시스템과의 완벽한 연동을 통한 정보의 일관성 유지
- 중복 투자의 위험 요소의 배제를 통한 정보의 가치성 부여
3. 유무선 통합 솔루션 제공을 통한 시.공간의 제약 극복
- Wireless의 영역을 유선의 영역까지 확대함으로써 시.공간의 제약에서 탈피
4. DSS(Decision Support System) 지원
- 빠른 의사결정 속도와 신뢰성 향상
- 문서 종류별, 폴더별, 등록현황 및 평가현황 등 다양한 통계 정보 제공

주요 기능

1. Personalized Enterprise Knowledge Portal
- 개인의 편의성을 고려한 기능 제공
- 개인별로 특화된 업무제공을 통한 편의성 제공
2. Portal Infra 환경 제공
- 통합 View환경의 제공
- 통합 검색 환경 제공
3. Knowledge Service
- 지식생성 / 분류 / 평가 /관리 지원
- 커뮤니티별 지식공유 기능
- 지식경영을 위한 기반 인프라 활용
4. Community Service
- 동호회 별, Project 별 커뮤니티 지원 기능
- 정규조직 외에 다양한 형태의 비정규적인 조직 지원 기능
- Community별 특화된 서비스 및 화면 기능 제공
5. Collaboration Service
- 지식정보의 공유 기능
- 화상회의, 게시판 활요 기능
- 메일기능 / 전자결재 기능, 일정관리, 명함등록 등
6. Notification Service
- Event 알림 기능
7. Knowledge Map/Repository Service
- 지식을 Map 형태로 유지하여 용이한 지식 탐색 기능 제공
- 메타 데이터 관리 기능
8. KeyDocs
  1) 문서관리
- 문서 Check In/Check Out 기능
- 문서 등록/삭제 기능
- 지정된 문서의 보안 설정 기능
- 폴더 등록, 변경, 삭제, 복사, 이동 기능 문서에 대한 버전 정보 기능
- 복합 문서의 첨부 기능
- 문서에 대한 인덱스 설정/해제 기능

  2) 컨텐츠 관리
- Template: 웹 사이트에 적용할 홈페이지 포멧
- Converter: 기업 내외에서 발생된 문서를 전달 받은 파일을 웹으로 게시하기 위해 전환하는 변환 기능
- Workflow: 문서에 대한 결재 모듈을 통해 웹으로 게시한다.
- Scheduler/Publishing: 변환 문서를 스케쥴러와 퍼블리셔를 통해 웹 게시
- Multi-Channel Service: 유무선 장비들과의 연동을 통한 컨텐츠 서비스
