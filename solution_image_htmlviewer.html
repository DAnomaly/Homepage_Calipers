HTML Viewer

 HTML Viewer는 풍부한 웹 문서 보기 및 관리 응용 프로그램을 제공하는 솔루션입니다.
 Tripleaims의 HTML Viewer를 통해 웹 작업 환경에서 다양한 확장자를 가진 문서들을 ActiveX없이 조회할 수 있는 환경을 제공합니다.

 DOCX, TXT, PDF, SVG, 메타파일(WMF, EMF) 레스터 이미지(TIFF, JNG, JBIG2, JPEG2000, WEBP etc.)을 포함한 100개 이상의 파일 형식을 지원하며, 여러 확장자를 가진 문서를 서버에 보관하고 조회하는 데에 불편함이 없도록 합니다. 또한, 지원하는 확장자에 대해 PDF 및 TIFF로 저장할 수 있습니다.

 Tripleaims에서 제공하는 솔루션과 함께 활용함으로써, 전자서류의 조회 및 등록에 또한 연계될 수 있게 하였습니다. 서버에 있는 전자서류를 조회하고 등록할 수 있도록 구현하여 보다 나은 이미지관리 프로그램을 활용할 수 있도록 솔루션을 제공합니다.

◯◯보험 적용사례

주요 특징

1. HTML5환경으로 제공
- HTML5 브라우저를 통해 모든 플랫폼 및 장치(PC, 모바일 등)에서 작동
- 문서를 썸네일로 생성하여 보다 편리한 조회
- 추가적인 ActiveX 설치 없이 HTML5환경으로 제공
2. 풍부한 파일 형식
- 지원되는 100개 이상의 다양한 확장자
- DOCX, TXT, PDF, SVG, 메타파일(WMF, EMF) 레스터 이미지(TIFF, JNG, JBIG2, JPEG2000, WEBP etc.) 등
3. 유니코드
- 지원되는 파일 형식에 대해 PDF 또는 TIFF로 저장 기능 지원

주요 기능

1. 문서 조회
- 서버에 저장되어 있는 문서를 인터넷 브라우저를 통해 검색하고 조회
- ActiveX를 설치하지 않고 다양한 확장자를 가진 문서를 이미지로 조회
2. PDF 또는 TIFF로 저장&인쇄
- 지원되는 파일 형식에 대해 PDF 또는 TIFF 저장 기능 지원
- 조회한 문서에 대해 전체 인쇄, 한 페이지만 인쇄 지원
3. 브라우저 환경을 제공하는 모든 플랫폼 및 장치에 지원
- PC, 모바일 등의 환경에서도 지원
