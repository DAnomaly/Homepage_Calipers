고속전송 프로그램

 고속전송 프로그램이란, 파일을 전송하는데 있어 속도와 보안 그리고 신뢰성을 제공하는 소프트웨어 솔루션입니다.

 FileCatalyst의 UDP 및 TCP 기술을 이용한 빠르고 안정적이며 유연한 파일 전송을 제공하며 SSL을 통해, AES를 이용하여 데이터 전송 암호화를 통해 보안성을 향상시켰습니다.
 해당 솔루션을 이용하시면 열악한 네트워크 환경에서도 때로는 FTP 보다 1000배 이상 파일전송 속도를 보여줍니다. UDP 기반 파일 전송 기술을 이용하여, 패킷 손실과 지연의 영향을 받지 않고, 최대 10Gbps 전송 속도로 파일을 전송하는 초고속 파일 전송 솔루션입니다.

 다수의 이미지 파일을 전송 및 관리한다는 것은 종종 많은 용량의 데이터를 전송해야 하며 이 작업은 인터넷 환경에 따라 오랜 시간이 걸리기도 합니다. 고속전송 프로그램 솔루션을 통해 느린 전송 문제를 해결하고 더 나은 이미지관리와 작업환경을 제공할 수 있습니다.

기존의 인터넷 전송 방식(TCP)
고속전송 프로그램(FileCatalyst : UDP)

주요 특징

1. 전송 중 파일 보안
- SSL을 통해, AES를 이용하여 데이터 전송 암호화가 가능합니다.
2. 파일 전달 보증
- 실시간 또는 MD5 checksums를 이용하여 각 파일을 검증합니다.
3. 성장 파일 전송
- 인코딩이 되는 동안 파일이 생성완료 될 때까지 기다림 없이 파일 전송 기능
4. 파일 버전 관리
- 기존 파일에서 달라진 부분만 전송하는 기능. 변경된 바이트만 전송 기능
5. 압축
- GZIP 또는 LZMA 알고리즘을 이용하여 데이터를 압축하여 전송 기능
- 파일을 전송하는 즉시 압축하여 압축 및 해제 프로세스를 줄이고 대용량 데이터 시트를 빠르게 전송합니다.

주요 기능

1. 업무 효율 보장
- 열악한 네트워크 환경에 최적화, 최대 10Gbps 고속 데이터 전송
- 대용량 또는 대량의 파일 전송에 효과적
2. 안정성 및 보안성 향상
- 연결 및 대역폭 관리, 전송 보장 및 우선순위 기능, 패킷 손실 면역화 및 암호화 고속전송
3. 유연한 확장성과 통합성
- 다양한 운영체제 지원, 솔루션의 모듈화
4. 커스터마이징
- SDK 옵션으로 기존 응용프로그램에 FileCatalyst 기능 통합 가능

적용 범위

1. 데이터 미러링
- 데이터 서버와 복구 서버의 동기화 기술에 고속전송 프로그램을 이용하여 재해에 대해 대비할 수 있습니다.
2. 데이터 배포
- 가속화된 속도로 필요로 하는 파일을 전송하거나 요청할 수 있습니다.
- 가속화된 속도로 팀 구성원 및 동료에게 콘텐츠를 배포하거나, 최종 사용자에게 제품 및 업데이트를 배포할 수 있습니다.
3. 클라우드 스토리지 통합
- 클라우드 스토리지를 활용하여 여러 공급자의 스토리지를 통합하여 관리할 수 있습니다.
