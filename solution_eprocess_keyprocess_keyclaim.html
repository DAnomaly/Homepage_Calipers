KeyClaim
- 사고 보상 심사업무 자동화 솔루션

키클레임(KeyClaim)은 보험회사의 사고보험금 지급과 같은 보상업무 프로세스를 효율적으로 수행하기 위하여 이미지 시스템과 워크플로우 시스템을 기반으로 보상업무자동화를 구현한 솔루션입니다.

사고 보상 신청 문서의 접수, 분배, 사고조사, 심사, 관리에 이르는 보상업무 프로세스를 재정의 하고, 자동화하여 심사자에게 정확한 판단을 내릴 수 있도록 지원하며, 빠른 고객지원 서비스가 가능하도록 합니다.

키클레임 도입 후 업무 흐름도

주요 특징

1. 검증된 시스템
- EDMS 분야의 최대의 Market Share를 자랑하는 FileNet의 엔진을 기본으로 탑재한 검증된 시스템
2. 시스템 확장의 용이성
- 작업의 과부하, 네트워크의 과부하에 따른 분산업무 환경 및 Multi Server, Multi System 구현
- 전자문서관리, COLD, 이미징, 워크플로우 제품을 단일 또는 복수로 구축 가능하고 향후 확대 시
- 유연한 통합 가능
3. 다양한 형태의 문서 수용
- Scan된 이미지, 동영상, 사진, Word, Excel, 녹취 데이터까지 종류에 상관없이 저장 관리
4. 대용량 고속 처리
- 1일 백만건 이상의 Transaction을 처리 할 수 있는 유일의 전문 금융 솔루션
- 처리의 긴급성에 따른 우선 순위 처리 기능
5. 완벽한 분산성 및 무정지 시스템 관리
- 완벽한 분산 서버 지원은 물론 원격지 제어 및 서비스
- 시스템 가동 중 설치 및 백업으로 무정지 시스템 관리 가능
6. 완벽한 백업과 Recovery
- 온라인 무정지, 백업 광디스크 자동 복제 및 복구
7. 업무 변경에 따른 신속한 적용
- 업무 변경 시 Process Map 수정(Visual Map)에 의한 변경업무 적용
8. 자동심사 강화로 인한 효율적인 업무 진행
- 심사 종류를 전문가 및 일반 심사 등으로 분류하고 일반 심사 중 조건을 다양하게 입력하여 자동
심사 처리를 적극 활용함으로써 효율적인 업무를 할 수 있도록 함

주요 기능

1. 입력기능
- 사고 보상 신청서와 관련 서류의 이미지 스캐닝
- OCR/OMR 인식 기능
- FORM Processing
- 이미지 인덱싱
- 데이터 압축, 즉시/배치/예약 전송
2. 자료 분배 기능
- 결재(결재/보류/반려/전결): 지급 승인자 결정 및 승인
- 접수~승인에 이르는 이력 관리
- 자필 서명 비교
- 자동심사 및 전문가 심사 등 단계별 심사 기준 적용
- 지점별, 보종별 심사 배정
3. 조사기능
- 사고 조사자 중 보완서류가 필요한 경우 즉시 추가
- 사진 및 피해정도 자료 첨부
- 고객 작성 신청서 및 사진 자료 즉시 송부
4. 심사기능
- 상태별 또는 신청일자별로 WF 심사대상을 검색 조회
- 심사 대상인 고객의 정보 확인
- CTI 시스템인 PDS(Predictive Dialing System, 예측 다이얼링 시스템)와 연동으로 자동 전화 연결 기능
- 현재 진행되는 심사고객의 전체 심사건 확인
5. 통계 및 검색 기능
- 업무별, 유형별, 지점별, 처리 이력별, 호스트 거래 조회
- 다중 조건 별 검색
- 보정자, 심사자 평가 통계
- 모든 형태의 문서를 어떤 환경에서도 접근 가능
- 계층적 분류에 따른 폴더, 본문 내용, 제목 등 강력한 문서 검색 기능
6. 서버 모니터링
- 에이전트 서비스 모니터링
- MRII 서비스 모니터링
7. 관리자 기능
- 사용자 관리, 코드 관리, 심사 자료 관리, 심사 기준 관리
- 심사자 관리(심사자 변경 및 Queue 이동)

적용 범위

1. 업무 프로세스 변경 및 설계 시

신체손해배상특약부화재보험, 화재보험, 주택화재보험, 재산종합보험,
동산종합보험, 기계보험, 건설공사보험, 운송업무, 조립보험

2. 각종 서류(신청서 및 청약서) 및 첨부문서의 이미지화

입회신청서, 사고보험지급 청구 서류, 법적조치 서류, 특별 인수 심사,
사고보험 조사/지급 심사
